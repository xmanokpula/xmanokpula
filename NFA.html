Non-Deterministic Finite Automata

Non-Deterministic Finite Automata (NFA) - from the word non-deterministic, this means that it can be in, and transition to, "multiple states" at once depending on the given input. In NFA, there might be several possible final states that could result from reading a given string of inputs; as long as at least one of them is a final state, then the string is accepted.

Now that we know a NFA may transition to multiple state at a time, then it is obvious that the transition function of a NFA is different from DFA which can be defined as δ: Q x ∑→ 2 ^ Q. This means that we take the result state of a transition from the power set of Q, not just from the element of set Q itself. Power set means "the set of all subsets of a given set", for example we have Q = {A,B,C}, the Power set of Q or P(Q) = {{∅},{A},{B},{C},{A,B},{A,C},{B,C},{A,B,C}}.

NFA and Epsilon - the epsilon (ε) means a state going to another state without any input. This can only happen if the current state has ε-transition to another state/s. By default, every state has an ε-transition on itself. ε-closure of a state or ε-closure(Q) is the set of all ε-transition or the set of all states where Q can transition to without any input.

Graphical Representation of NFA

  1. The states are represented by vertices (circles).
  2. The arc or arrow labeled with input/s or ε show the transitions.
  3. The initial state is marked with plain arrow.
  4. The final state/s are marked by double circle.
